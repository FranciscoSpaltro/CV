Latest Works

Development and Implementation of an Onboard Computer for a CubeSat Satellite Using a Real-Time Operating System and Open-Source Libraries

This project is being developed together with my two teammates as part of our Professional Practical Work, a requirement for obtaining a degree in Electronic Engineering. It is part of a broader initiative led by the Faculty of Engineering at the University of Buenos Aires, aiming to develop and launch a CubeSat, a small-sized satellite designed to experiment and advance in space technology.

In this context, we focus on implementing a functional prototype of the Onboard Computer (OBC). The main goal is to develop a standardized platform that not only facilitates future iterations of the satellite but also serves as an educational and technical tool for developing skills in the aerospace field.

The project includes researching different microcontroller architectures compatible with CubeSat literature and project-specific requirements, developing various communication protocols essential for interaction between the satellite's subsystems, emphasizing reliability and efficiency for aerospace applications, and building a CI/CD (Continuous Integration - Continuous Delivery) test environment to automate and streamline validations and updates for the OBC, ensuring proper operation and continuous adaptation to new missions or satellite requirements. We use FreeRTOS as the operating system, libopencm3 as an open-source library, and GitLab CI/CD along with a Raspberry Pi 5 for automated testing.

Capacitance Meter

Development of an ASM program that calculates the value of an unknown capacitor and displays the obtained value and its charge curve on a PC. This project was part of the final integrative project for the Microprocessors Laboratory course (FIUBA). The program was recorded on an Atmega328p microcontroller using an Arduino UNO development board. The system generates a 5V pulse to charge the capacitor and acquires voltage values using the ADC until it reaches 63.3% of its charge. TIMER 1 is used to measure the charge time, and the capacitor value is then calculated. The measured values and calculated capacitance are transmitted via serial communication.
